# AMS-595-Project-7
A series of C++ code.
The first code, insert.cpp, asks for a number input and returns if the value was one, zero, negative one, or another value.
The second, print_vector.cpp, takes a given vector, converts the elements into strings, and prints them.
while_loops.ccp returns the numbers in the Fibonacci Sequence using a while loop.
The next code, isprime.cpp: tests whether a given number is prime by counting its factors.
factor.cpp prints all factors of a given number.
prime_factorization.cpp returns all prime factors of a number.
pascals_t.cpp asks for a number input and returns that many rows of Pascal's triangle
